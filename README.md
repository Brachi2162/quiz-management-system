# Comprehensive Quiz Management System

A production-ready, object-oriented Quiz Management System developed in Java. The application delivers a robust architecture for managing users, designing interactive quizzes, and tracking performance metrics efficiently.

## Core Features

*   **User Authentication & Management:** Secure registration and login flows with role-based access control.
*   **Dynamic Quiz Engine:** Full capabilities to manage question banks, support multiple-choice formats, and handle quiz sessions.
*   **Automated Evaluation:** Instant scoring logic, answer validation, and performance summary generation.
*   **Data Integrity:** Structured data handling ensuring clean separation between user profiles and quiz statistics.

## Architecture & Design Principles

*   **Object-Oriented Programming (OOP):** Strict adherence to encapsulation, inheritance, and polymorphism for maximum code reusability.
*   **Clean Code Architecture:** Clear separation of concerns between data models, business logic, and user interaction layers.
*   **Scalability:** Designed with modular interfaces allowing easy integration of database storage or graphical user interfaces in the future.

## Prerequisites

*   **Java Development Kit (JDK):** Version 17 or higher.
*   **Integrated Development Environment (IDE):** Any modern IDE supporting Java (such as IntelliJ IDEA, Eclipse, or VS Code).
